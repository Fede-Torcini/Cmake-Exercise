Compiling


1 Compile with Batch file.
	1.1 Make sure "exercise_compile.bat" is next to src folder
	1.2 Run Batch file

2 Compile with cmake
	2.1 Create a build folder in the same directory as the "src" folder 
		ex: "c:ProjectDir\cmake_ejercicio\src\" create "c:ProjectDir\cmake_ejercicio\build\"

	2.2 From the build directory run 
		cmake ../

	2.3 From the build directory run 
		cmake --build .
